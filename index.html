Day 2

Find:
  • Find out how to access files with and without code blocks. What is the benefit of the code block?
    • Use File.new() to access file without a code block, File.open() takes a code block
    • It seems using a code block allows you to process the file line by line. Using File.open seems to be a bit more granular
    • Great examples: http://pleac.sourceforge.net/pleac_ruby/fileaccess.html
  • How would you translate a hash to an array? Can you tranlate arrays to hashes?
    • One way: 
      >> array = ["stay", "classy", "san", "diego"]
=> ["stay", "classy", "san", "diego"]
>> hash = Hash[*array]
=> {"san"=>"diego", "stay"=>"classy"}
    • I can imagine there are more pragmatic needs than this: 
      >> back_to_array= Array[*hash]
=> [["san", "diego"], ["stay", "classy"]]
  • Can you iterate through a hash?
    • I think I know how to do this! 
      >> hash.each {|key, value| puts "key: #{key}, value: #{value}"}
key: san, value: diego
key: stay, value: classy
  • You can use Ruby arrays as stacks. What other common data structures do arrays support?
    • It supports queue by using the 
      shift
      and 
      unshift
      operators
    • I'm not sure of other native data structures, but many can be implemented by extension
    • See Manipulating Structured Data in Ruby (Hal Fulton, InformIT)
